# 记忆系统快速参考

## 🎯 核心功能

助手会自动记住你告诉它的重要信息，并在后续对话中使用这些信息提供个性化服务。

## 📝 如何让助手记住信息

### 方式 1：明确说明（推荐）

```
请记住，我的名字是张三
请记住，我喜欢喝咖啡
请记住，我在北京工作
```

### 方式 2：手动添加

```
/memory_add 我的生日是 3 月 15 日
/memory_add 我使用 Python 和 Go
```

## 🔍 记忆管理命令

| 命令 | 说明 | 示例 |
|------|------|------|
| `/memory` | 查看所有记忆 | `/memory` |
| `/memory_add <内容>` | 手动添加记忆 | `/memory_add 我喜欢喝咖啡` |
| `/memory_search <关键词>` | 搜索记忆 | `/memory_search 咖啡` |
| `/memory_delete <ID>` | 删除指定记忆 | `/memory_delete user_123_1234567890` |
| `/memory_clear` | 清空所有记忆 | `/memory_clear` |

## 💡 使用技巧

### ✅ 会被记住的信息

- 个人信息：姓名、年龄、职业、所在地
- 偏好习惯：喜欢/不喜欢什么
- 工作内容：技能、项目、工作方式
- 明确要求记住的事情

### ❌ 不会被记住的信息

- 普通闲聊
- 一次性问题
- 临时性的信息

## 🔧 故障排查

### 如果助手没有记住信息

1. **使用手动添加**（最快）
   ```
   /memory_add 我喜欢喝咖啡
   ```

2. **明确说"请记住"**
   ```
   请记住，我的名字是张三
   ```

3. **查看日志**
   - 运行 bot 时查看控制台输出
   - 查找 "保存新记忆" 或 "提取记忆失败" 等信息

4. **检查记忆文件**
   ```bash
   cat bot/data/memories.json
   ```

5. **运行测试**
   ```bash
   python test_memory_extraction.py
   ```

### 常见问题

**Q: 为什么有些对话没有被记住？**

A: AI 会判断信息是否值得长期记住。如果你认为应该记住，明确说"请记住..."或使用 `/memory_add`。

**Q: 记忆会自动更新吗？**

A: 目前不会。如果信息变化，需要先删除旧记忆，再添加新的。

**Q: 记忆有数量限制吗？**

A: 没有限制，但每次对话只会使用最近 10 条记忆。

## 🎨 记忆分类

记忆会自动分类，方便管理：

- 👤 **personal** - 个人信息
- ⭐ **preference** - 偏好设置
- 💼 **work** - 工作相关
- 📌 **fact** - 重要事实
- 📋 **other** - 其他

## 🚀 最佳实践

1. **定期检查记忆**
   ```
   /memory
   ```

2. **及时更新过时信息**
   ```
   /memory_delete <旧记忆ID>
   ```
