# 1.1.15

本版重点补齐 CLI 子 agent 集群工作流、桌面智能体管理台和聊天工作台细节。

## 智能体和集群

- CLI Bot 现已支持更完整的子 agent 管理；非集群模式可直接切换 active agent，集群模式可在消息里用 `@agent_id` 分发。
- 新增桌面智能体管理页，可统一维护 CLI、工作目录、子 agent 和 CLI 参数。
- 集群模式补齐启用开关、MCP 状态、模型档位、模板预设和 JSON bundle 导入 / 预览 / 覆盖应用。
- Bot 列表和切换入口现在会正确汇总主 agent 与子 agent 的忙碌状态，不再只显示单一 agent。

## 聊天和工作台

- Chat 输入框增加子 agent mention 联想和补全，相关历史、会话和 activity 状态都按 agent 作用域隔离。
- 文件树补了选中态和更多工作台会话同步细节，桌面工作流更稳定。
- Settings 和 Desktop Workbench 做了收口调整，智能体相关维护入口更集中。

## 后端和 Assistant

- cluster backend 增加 runtime、setup、bundles 和模板配置，Web API / manager / real client 一并接入。
- assistant 相关模块整理到 `bot/assistant/` 子目录下，`cron`、`dream`、`memory`、`upgrade` 等结构更清晰。
- Web API、cluster、assistant 和前端 real client 补了更多回归测试。

## 文档

- README 更新为当前产品态说明，补了托管 Bot、assistant 自动化和 `managed_bots.json` 示例。
- AGENTS / CLAUDE guide 同步刷新，补了 session 键、Assistant Ops 和当前测试入口说明。
