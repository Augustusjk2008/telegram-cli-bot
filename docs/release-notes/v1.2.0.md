# 1.2.0

本版重点补强桌面工作台动效和智能体管理细节，补齐常用维护闭环。

## 智能体和集群

- 新增智能体时，工作目录选择器支持直接新建文件夹，不用离开弹窗。
- 删除智能体时，可选同时删除历史记录，范围覆盖该 Bot 下所有子 agents 历史。
- 删除历史按 bot 维度精确清理，共用同工作目录的其它 Bot 不受影响。

## 聊天和工作台

- 桌面工作台补了成组微动效，包括快速打开、发送反馈、流式输出提示、Trace 展开、状态栏运行提示等。
- 智能体管理和工作目录选择流程做了收口，常见操作路径更短。

## 后端和 Assistant

- Web API 新增删除 Bot 时联动清理历史的能力，并同步清掉本地会话残留。
- chat store 增加按 bot 删除历史能力，覆盖主 agent 和子 agents。
- 前后端 real client、桌面管理页和历史存储补了回归测试。
