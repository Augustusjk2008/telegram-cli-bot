# 记忆系统故障排查指南

## 问题：助手不记住我告诉它的事情

### 可能原因

1. **AI 记忆提取未触发**
   - 记忆提取是异步后台任务，可能因为异常而失败
   - AI 可能判断对话内容不值得记住

2. **JSON 解析失败**
   - Claude API 返回的 JSON 格式不正确
   - 响应包含额外的文本或 markdown 代码块

3. **API 配置问题**
   - ANTHROPIC_API_KEY 未设置或无效
   - ANTHROPIC_BASE_URL 配置错误（如果使用代理）

### 解决方案

#### 1. 检查日志

运行 bot 时查看日志输出，关注以下信息：

```bash
python -m bot
```

查找日志中的关键信息：
- `开始提取记忆: user_id=xxx` - 记忆提取已触发
- `✅ 保存新记忆成功` - 记忆保存成功
- `未提取到值得记住的信息` - AI 判断无需记住
- `❌ 提取并保存记忆失败` - 提取过程出错

#### 2. 手动添加记忆（临时方案）

如果自动提取不工作，可以手动添加记忆：

```
/memory_add 我叫张三，是一名软件工程师
```

#### 3. 测试记忆提取功能

运行测试脚本验证记忆提取是否正常：

```bash
python test_memory_extraction.py
```

这会测试 4 个场景：
- 明确的个人信息
- 偏好信息
- 普通闲聊（不应记住）
- 明确要求记住

#### 4. 检查 API 配置

确认 `.env` 文件中的配置：

```bash
# 必需
ANTHROPIC_API_KEY=sk-ant-xxx

# 可选（如果使用代理）
ANTHROPIC_BASE_URL=https://your-proxy.com/v1
ANTHROPIC_MODEL=claude-3-5-sonnet-20241022
```

#### 5. 查看记忆数据

检查记忆文件内容：

```bash
cat bot/data/memories.json
```

如果文件为空或只有 `{"version": "1.0", "memories": []}`，说明没有记忆被保存。

#### 6. 运行完整测试

```bash
python -m pytest tests/test_memory.py -v
```

所有测试应该通过。如果有失败，说明基础功能有问题。

### 技术细节

#### 记忆提取流程

1. 用户发送消息 → 助手回复
2. 后台异步调用 `extract_memories_from_conversation()`
3. 使用 Claude API 分析对话，判断是否有值得记住的信息
4. 如果有，提取精炼的记忆内容、分类和标签
5. 保存到 `bot/data/memories.json`

#### 记忆注入流程

1. 用户发送消息
2. 从 `memories.json` 加载用户的最近 10 条记忆
3. 将记忆注入到系统提示词中
4. 调用 Claude API 生成回复

#### 数据结构

```json
{
  "version": "1.0",
  "memories": [
    {
      "id": "user_123_1709712000",
      "user_id": 123,
      "content": "用户名叫张三，是一名软件工程师",
      "category": "personal",
      "created_at": "2026-03-06T10:00:00",
      "updated_at": "2026-03-06T10:00:00",
      "tags": ["姓名", "职业"]
    }
  ]
}
```

### 常见问题

**Q: 为什么有些对话没有被记住？**

A: AI 会判断对话内容是否值得长期记住。普通闲聊、一次性问题等不会被记住。如果你认为某些信息应该被记住，可以：
- 明确说"请记住..."
- 使用 `/memory_add` 手动添加

**Q: 记忆会自动更新吗？**

A: 目前不会。如果信息发生变化，需要：
1. 使用 `/memory_delete <ID>` 删除旧记忆
2. 重新告诉助手新信息，或使用 `/memory_add` 添加

**Q: 记忆有数量限制吗？**

A: 没有硬性限制，但每次对话只会注入最近 10 条记忆到上下文中。

**Q: 可以手动编辑记忆文件吗？**

A: 可以！`bot/data/memories.json` 是标准 JSON 格式，可以直接编辑。编辑后重启 bot 即可生效。

### 下一步优化方向

5. **记忆摘要** - 当记忆过多时，自动生成摘要
